# Risk Governance

- Risk engine has veto authority.
- LLM cannot override deterministic risk controls.
- Maximum portfolio drawdown assumption: 15%.
- Max position size to be defined later.
- Max risk per trade to be defined later.
- No trade if data is stale.
- No trade if liquidity is insufficient.
- No trade if strategy is not approved for the current phase.
- No live trading in current phase.
- Cash/no-trade is a valid output.
- Risk decisions must be logged.
