# Trading Research Agent Brief

## Purpose

EdgeLab is a local-first trading research and validation system. Its purpose is to discover, define, backtest, rank, paper-trade, monitor, and report on trading strategies before any real-money execution is considered.

## Initial User Goals

- Start with a $50,000 model portfolio assumption.
- Target attractive percentage returns with strong risk controls.
- Respect a maximum portfolio drawdown assumption of 15%.
- Focus first on US equities and ETFs.
- Begin with daily and multi-day swing strategies.
- Build repeatable evidence before considering any live trading path.

## Research-First Philosophy

EdgeLab treats every strategy as unproven until evidence supports it. A useful system must be comfortable saying no.

**No edge means no trade.**

The system should optimize for risk-adjusted return, robustness, and repeatable strategy evidence rather than fantasy monthly dollar profit projections.

## Future Direction

Paper trading is expected to use Alpaca first in a later phase. Interactive Brokers may be considered later. Cloud deployment should be possible later, but Phase 0 remains local-first.
