Local MarketData.app download helper

  1. Create a MarketData.app account.
  2. Get an API token.
  3. In Terminal, set export MARKETDATA_APP_TOKEN="[paste your MarketData.app token]".
  4. Run dry-run first: PYTHONPATH=src python -m edgelab.intraday.marketdata_app_downloader --symbols SPY CSGP --months 12 --dry-run
  5. Run the actual local download only when ready: PYTHONPATH=src python -m edgelab.intraday.marketdata_app_downloader --symbols SPY CSGP --months 12
  6. Confirm the CSV files are under data/raw/historical_intraday/firstratedata.
  7. Refresh this audit page.

Do not commit downloaded CSV files or API tokens. Use matching SPY and CSGP date ranges. If free daily credits are not enough, run smaller windows or upgrade only if you choose to.
